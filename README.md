## 🚀 Getting Started

### 1. Clone Repository

```bash
git clone https://github.com/abhinavsingh-hub/n8n-workflows.git
cd n8n-workflows
```

---

### 2. Run with Docker

```bash
docker-compose up
```

---

### 3. Open n8n

```
http://localhost:5678
```

---

### 4. Import Workflows

* Go to n8n UI
* Click **Import**
* Upload JSON files from `/workflows`

## 🔐 Environment Setup

Create a `.env` file:

```env

N8N_BASIC_AUTH_USER=admin
N8N_BASIC_AUTH_PASSWORD=yourpassword
```
